**Design language:** Utilitarian. Not military, not luxury — university-budget engineering. The kind of ship you'd build if you had a physics grant and no oversight. Angular but not aggressive. **Key features:**
- Oversized sensor antenna array (custom-built, conspicuously large)
- Cracked viewport on the bridge
- Docking collar on port side
- Fold drive housing visible as a bulge on the ventral hull
- Orange running lights (when powered)

## 10. WORKFLOW NOTES

### Recommended Generation Order

1. **Global style anchors** (Section 1) — Lock in the visual language
2. **DENT model sheet** — He's the most visually complex character
3. **The Vex model sheet** — The ship players spend the most time in
4. **Vin model sheet** — Player character
5. **Vex interior locations** — Bridge, Engineering, Quarters, etc.
6. **SIC ships and Graves** — Primary antagonist faction
7. **Folder ships and Kade** — Secondary faction
8. **Technology close-ups** — Torquer, fold drive, null cells
9. **Key scene illustrations** — The big moments
10. **Moodboards** — Fill in the gaps

### V7 Feature Usage

| Feature | When to Use |
|---------|-------------|
| `--oref` | ALL character and ship prompts after establishing the reference. This is how Vin looks the same in every scene. |
| `--ow` | Lower (25-50) for wildly different scenes. Higher (75-100) for consistency. Default 75. |
| `--sref` | Lock in after establishing style anchors. Apply to ALL subsequent generations for visual consistency. |
| `--style raw` | Ship interiors, technical details, prop designs — anything that needs literal prompt adherence. |
| `--s` (stylize) | Low (50) for model sheets and reference. Medium (150-200) for hero shots. High (250-300) for dramatic scenes. |
| `--p` | Enable after tuning your Global Profile to the game's aesthetic. |
| Strong variation | Early exploration of new designs. |
| Subtle variation | Refining a design you're 90% happy with. |
| Relax mode | Bulk environment generation, NPC portraits. |
| Fast/Turbo | Hero moments, key characters, cover art. |

### Prompt Modifiers for Consistency

Add these to the end of prompts as needed:

- **For grounded realism:** `photorealistic, practical effects, 35mm film grain`
- **For dramatic atmosphere:** `cinematic lighting, anamorphic lens, Ridley Scott cinematography`
- **For technical detail:** `macro photography, studio lighting, product photography`
- **For emotional moments:** `shallow depth of field, warm practical lighting, intimate composition`
- **Against AI-generic aesthetics:** `NOT cyberpunk, NOT neon, NOT clean futuristic, worn industrial practical`

### Aspect Ratios

| Use | Ratio |
|-----|-------|
| Model sheets / turnarounds | `--ar 3:2` |
| Character portraits | `--ar 2:3` |
| Ship/environment hero shots | `--ar 16:9` |
| Ultra-wide establishing shots | `--ar 21:9` |
| Detail close-ups | `--ar 4:3` or `--ar 1:1` |
| Vertical dramatic moments | `--ar 9:16` |
